# Entrega-GPs.
Un programa donde se muestran diversas funciones para simular Procesos Gaussianos (GPs) con distintos kernels.

Un Proceso Gaussiano es un tipo de Proceso Estocástico que se caracteriza porque cualquier distribución finito-temporal $(X_{t_1} \ldots X_{t_n})$ se comporta conjuntamente como una normal multivariante.

Los kernels que aparecen son los siguientes:
Kernel RBF (Radial Basis Function)
Kernel periódico
Kernel DPK polinómico (Dot Product Kernel)
Además, aparecen implementadas funciones que simulan GPs con
Kernel arbitrario (A dar por el usuario)
Kernel DPK arbitrario (A dar la familia de funciones por el usuario)

También aparecen algunas visualizaciones (gráficas).
